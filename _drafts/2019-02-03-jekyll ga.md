대부분의 포털 블로그를 이용하면 today 방문자수나, 검색유입 키워드 등이 기본 제공된다. 하지만 jekyll로 블로그를 운영하면 그런 통계는 얻을 수가 없는데 이 글에서는 google-analytics를 이용한 통계를 가져오는 방법을 포스팅한다.

Google analytics는 연동되어 있다고 가정하고 진행한다.

### 1. Google analytics API 설정하기

<a href="https://ga-dev-tools.appspot.com/query-explorer/">https://ga-dev-tools.appspot.com/query-explorer/</a>에 접속해서 주황색 부분의 "click here to authorize" 부분을 클릭하고 로그인 한다.

![ga devtool main]({{"/images/posts/jekyllga-1.png"| relative_url}})

### 2. 통계 조회 쿼리 옵션 선택하기

로그인을 했다면, Select a view 부분에 지킬 홈페이지를 선택한다.<br/>
그다음은 Set the query parameters 부분을 설정할 차례다.<br/>
일별 데이터를 얻어올 것이므로, start-date와 end-date를 오늘 날짜로 둘다 설정한다. metrics는 sessions와 pageviews를 가져올 거니까 둘다 선택하자. 빈 공간을 클릭하면 여러개 추가할 수 있다. dimensions에는 souce와 keyword를 추가해준다.<br/>
여기서는 예시로 든 것이고, 본인의 입맛에 맞는 데이터를 가져오면 된다.

![ga devtool setting]({{"/images/posts/jekyllga-2.png"| relative_url}})

### 3. 쿼리 실행하기

2번에서 세팅 후에 아래쪽 Run Query버튼을 누르면, API Query URI를 얻을 수 있는데, 밑에있는 Include current access_token in the Query URI를 체크해준다.

![ga devtool setting]({{"/images/posts/jekyllga-2.png"| relative_url}})
