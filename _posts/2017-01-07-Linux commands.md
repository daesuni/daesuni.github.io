title: Linux/리눅스 자주 쓰는 명령어 정리
---

실무에서 자주 쓰는 Linux/리눅스 명령어를 설명과 함께 정리한다.

- **pwd**: 현재 위치한 경로 표시
- **cd**: 현재 위치한 경로 이동
- **ls**: 현재 폴더의 폴더/파일들의 이름 표시
- **ls -l**: 현재 폴더내 폴더/파일들의 이름, 퍼미션, 파일수 등 상세정보 표시
- **ls -al**: 현재 폴더내 폴더/파일 (숨김파일 포함) 들의 이름, 퍼미션 등 상세정보표시
- **cp**: 폴더/파일 복사
- **cp -r**: 폴더/파일 및 하위경로까지 복사
- **mv**: 폴더/파일 이동
- **rm**: 폴더/파일 삭제 (삭제전 확인)
- **rm -r**: 폴더/파일 및 하위경로까지 삭제
- **mkdir**: 폴더 생성
- **cat**: 파일의 내용을 화면에 출력
- **more**: 화면 단위로 파일의 내용을 출력
- **find**: 폴더/파일 검색
- **find -name**: 특정 이름의 폴더/파일 검색
- **touch**: 크기가 0인 파일 생성
- - -
- **chown [소유자:소유그룹] [폴더/파일명]**: 폴더/파일의 소유자 변경
- **chmod [옵션] [퍼미션] [폴더/파일명]**: 폴더/파일의 퍼미션 변경
- **chgrp [옵션] [그룹] [폴더/파일명]**: 폴더/파일의 소유 그룹 수정
- - -
- **tar -cvf [압축파일.tar] [폴더/파일명]**: tar로 압축하기
- **tar -xvf [압축파일.tar]**: tar로 압축풀기
- **tar -zcvf [압축파일.tar.gz] [폴더/파일명]**: tar.gz로 압축하기
- **tar -zxvf [압축파일.tar.gz]**: tar.gz로 압축풀기
- **gzip [폴더/파일명]**: gz로 압축하기
- **gzip -d [압축파일.gz]**: gz로 압축풀기
- - -
- **yum**: 인터넷을 연결해서 rpm 패키지 설치
- **yum install nano -y**: Nano 에디터 설치

- **crontab**: 반복작업 관리
- **free**: 시스템 메모리 정보 출력
- **ps**: 현재 실행되고 있는 프로세스 목록 출력
- **pstree**: 프로세스의 정보를 트리 형식으로 출력
- **top**: 실시간으로 시스템 상황 모니터링
- **killall [프로세스명]**: 특정 이름의 프로세스를 모두 종료
- **kill -9 [PID]**: 특정 PID를 가진 프로세스 종료

- **shutdown**: 시스템 종료
- **poweroff**: 시스템 종료
- **reboot**: 시스템 재부팅
- **shutdown -r now**: 시스템 재부팅

- **ping**: IP 네트워크를 통해 특정한 호스트가 도달할 수 있는지의 여부를 테스트
- **ifconfig**: 리눅스 네트워크 인터페이스 설정 확인/관리
- **netstat**: 네트워크 접속, 라우팅 테이블, 네트워크 인터페이스의 통계 정보를 보여주는 도구
- **traceroute**: 네트워크를 통해 목적지에 도달하는 경로를 수집하는 리눅스 명령어
- **route**: 리눅스에서 route 설정 상태 확인하는 명령어
- **clock**: 하드웨어 시계 조회/설정하는 리눅스 명령어
- **date**: 시스템 날짜, 시간을 출력 또는 변경하는 리눅스 명령어
