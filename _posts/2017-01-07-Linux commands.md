| 설명									| 명령어									|
| ------------------------------------- | ------------------------------------- |
| 파일 삭제 (확인)						| rm [파일명]							|
| 파일 삭제 (확인안함)						| rm -f [파일명]							|
| 폴더 생성								| mkdir [폴더명]							|
| 폴더 삭제								| rm -r [폴더명]/						|
| 소유자 변경								| chown [소유자:소유그룹] [폴더명]			|
| NANO 설치								| yum install nano -y					|
